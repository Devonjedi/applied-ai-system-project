# Assets

Store system architecture diagrams, walkthrough screenshots, and other visual artifacts here.

- Mermaid source for the architecture diagram lives in `system_architecture.mmd`.
- A rendered submission image already lives in `system_architecture.png`.
- If you want to revise the flow, edit `system_architecture.mmd` or `system_architecture.svg`, then re-export `system_architecture.png`.
- Save demo screenshots or GIF captures here so they can be referenced from `README.md`.
